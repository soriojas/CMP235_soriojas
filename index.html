What is MCC?

Description

MCC is a Gacha game created by Ambition and released for iOS and Android in 2013. Gacha is a genre of Japanese mobile games based on luck where players have a chance to roll in-game items of some value.

Development of other games of the same genre.

Ambition also created other games that used the same concept as MCC such as DG.

DG was the first mobile app that Ive ever known to use L2D graphics, and it wasnt until later on that other game developers started using L2D
